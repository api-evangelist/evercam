# EvercamThis is a repo for managing machine readable artifacts for the [Evercam APIs](http://www.evercam.io/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Evercam APIs](http://www.evercam.io/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.evercam.io/)- [Developer](http://www.evercam.io/)- [Twitter](https://twitter.com/evrcm)- [GitHub](https://github.com/evercam)The [APIs.json](https://github.com/api-evangelist/evercam/blob/master/apis.json) index provides links to all the relevant links for the Evercam APIs, including the OpenAPI and Postman if they are present.
